## 基础名词

- **PC(Personal Computer)**: 个人计算机。使用这个词的时候通常特指你的台式/笔记本电脑，用来与手机/平板区分。
- **操作系统（Operating System, OS）**：作为计算机基石的软件，电脑的“大管家”。它负责管理、执行其他软硬件，以及与使用者交互。常见 PC 端操作系统包括 Windows, Linux，MacOS 等，常见手机操作系统包括 Android、IOS 等。其他应用软件需要在特定的操作系统上运行，通常要为不同系统提供不同的版本。
- **内存(Memory)、存储/外存(Storage)**：Memory（标准翻译“内存”，港澳台翻译“记忆体”）是一个容量相对较小、断电之后所有数据都会丢失、用来暂存程序运行时数据的空间。Storage（存储/外存）则是用来永久存储数据的空间。
  - 由于早年智能手机行业在营销中有意或者无意的混淆，很多人会把后者称为“内存”——“我手机内存不够，装不了游戏，等我删几个软件”，而为了指代前者，再生造了一个“运行内存/运存”的概念。
  - 个人计算机中使用的存储介质通常为**硬盘/磁盘（Disk）**，因此交流中通常直接说“硬盘”或“磁盘”。它又分为相对便宜、容量大、速度慢的机械硬盘(HDD)和较贵、容量相对小、速度较快的固态硬盘(SSD)。
- **文件(File)**：计算机存储中存放数据或程序的基础单位。照片、视频、文档等数据都以一个个文件的形式存放在计算机中。
  - **目录/文件夹（Directory）**：这两个词是同义词，可以互换。一个文件夹中可能包含多个文件、也可以包含更下一层的文件夹。计算机中的所有文件/文件夹呈**树形结构**：将最上层的文件夹视为树根，它直接包含的每个文件/文件夹视为一个分杈，作为分杈的文件夹内部又会有更下一层的分杈，这样形成了一棵树。
  - **根目录（Root Directory）**：上一个概念中提到的作为“树根”的那个文件夹。计算机中通常会把硬盘分区使用（“C盘/D盘”），因此如果不加特别指明，说的就是某个硬盘分区的最上层一级。
  - **路径（Path）**：用来标识文件确切存储位置的一串字符。在 Windows 中，路径形如：`D:\高程\第一章课件.pdf`。上面这个路径表示在 D 盘根目录下、名为“高程”的文件夹中一个名称为“第一章课件.pdf”的文件。
  - **扩展名（Extension）**：文件名中最后一个点`.`之后的部分，用来标识文件的格式。例如，图片文件的扩展名可能是 `.jpg` 或 `.png` 等等，视频文件的扩展名可能是 `.mp4` 或 `.mkv` 等等。
    - Windows 通常会默认隐藏文件的扩展名，可能给初学者造成困扰。在环境配置一节中，我们会将其设置为显示全部扩展名。
    - 扩展名只是对文件格式的**标识**，如果你愿意，可以修改使用错误的扩展名或自己编造的扩展名（它会带来一些麻烦，但通常可控）。直接修改扩展名**不能**改变文件内容。如果你想要把文件从一个格式转为另一个格式（例如，老师要求提交特定格式的作业时），你应当使用专用的软件或在线工具。
  - **可执行文件(Executable)**：可以直接运行的文件，通常是一个程序。在 Windows 中，这类文件的扩展名为 `.exe`。
  - **压缩文件/压缩包(Archive)**: 一种特殊的文件，由若干个文件/文件夹打包形成，用于节省空间、方便传输。常见的压缩文件扩展名有 `.zip`、`.rar`、`.7z`。
    - Windows 11 新版的文件资源管理器中，可以直接双击打开常见的压缩包、查看里面的内容，就像打开普通文件夹一样，但需要注意的是，**不宜直接在压缩包中运行程序**，可能会导致意想不到的错误，应当先将压缩包**解压**之后再运行里面的程序。
    - 创建压缩文件需要下载安装专用软件，如 [WinRAR](https://www.win-rar.com/download.html) 或 [7-zip](https://sparanoid.com/lab/7z/) 等。`.rar` 格式为 WinRAR 的专有格式，其他压缩软件可以解压、但不能创建。
- **进程**：“运行中的程序”。同一个软件（如记事本）可以打开多份，每份都对应了一个进程，在任务管理器中可以看到多个同名的进程。当软件卡死、不能响应操作时，我们可以通过在任务管理器中强制结束进程来关闭它。少部分情况下程序未正常退出，即使窗口关闭了，进程仍然滞留在后台，也需要通过这种方式关闭。
