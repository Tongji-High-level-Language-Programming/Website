## 安装步骤（共 7 步）

### 1. 找到安装文件

从 QQ 群开学大礼包或 [此链接](/RedPanda-CPP-3.4.3279-With-gcc-15.2.0.zip) 获取**小熊猫 C++**的安装文件 `RedPanda-CPP-3.4.3279-With-gcc-15.2.0.zip`

- 本课程使用的是特别定制版本，不要另外从网上下载
  - **安装包名**: `RedPanda-CPP-3.4.3279-With-gcc-15.2.0.zip`
  - **安装包大小**: `572,489,651` 字节
- **注意**: 下面的安装步骤以定制版本为准，非定制版本后续不提供技术支持。
- 本课程提交的作业用定制版本编译，以定制版本的编译及运行结果为准，**版本不匹配造成的失分后果自行承担**。

### 2. 选择解压安装包

在 `RedPanda-CPP-3.4.3279-With-gcc-15.2.0.zip` 上按鼠标右键，选择“全部解压缩”

![解压](001-解压.png)

### 3. 将压缩包提取至用户文件夹

> [!IMPORTANT]
> 一定要严格按照操作步骤执行！

![配置解压位置](002-配置解压位置.png)

### 4. 等待解压完成

完成后，检查 `C:\Users\用户名` 中是否出现了 `RedPanda-CPP` 文件夹

![解压完成](003-等待解压完成.png)

### 5. 固定到开始菜单

进入 `C:\Users\用户名\RedPanda-CPP` 文件夹，找到 `RedPandaIDE.exe`，鼠标右键菜单选择"固定到开始。

![固定到开始菜单](004-固定到开始菜单.png)

### 6. 启动

打开"开始菜单"，选择小熊猫 C++ 并启动，启动成功则表示安装过程完成

![启动](005-启动.png)

**提示**: 如果想设置快速启动小熊猫 C++（固定到任务栏），参考 VS 的安装文档。

### 7. 检查默认选项

本课程整个学期的**所有作业**均要求这些配置项，因此建议定期检查。

:::
